# Exercise-Tracker-Microservice
Api microservice for tracking excercise

### Usage:
Get all users:  
```
GET /api/users  
```
Create a new user:  
```
POST /api/users  
```
Create exercise:  
```
POST /api/users/:_id/exercises  
```
Get user exercise log:  
```
GET /api/users/:_id/logs?[from][&to][&limit]  
```
